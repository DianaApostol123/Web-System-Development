Design of the application

The application is a web system able to properly assist a museum visitor. Basically, in order to have access to the museum's tour, user is invited to scan his ticket which contains a QR code. Once the QR code is

scanned, user has access to the application which provides information about the respective museum and its objects of culture. When the user stops in front of an object from the museum, the virtual tour also is interrupted until the user goes ahead. The application provides two languages for the user: romanian and english languages.Based on his preferences, user can set the language and the application will be translated.

Design of the application

The application contains 6 pages: Virtual tour, Photo album, Information, Contact, Review and Help & Support. Each page is detailed in the following rows.

Virtual Tour Page

This page will contain a section for scanning the QR code. In order for the tour to start, scanning of the QR code from the ticket must be done. After this, the tour will start. This will consist of an interactive

experience. The video and audio will have a common start. After that, depending on what area you are in the museum, the presentation will change accordingly. In order for the app to know the user positioning, the RFID technology will be used. There will be for each area of the museum an RFID transmitor. The app will read the data from the RFID and will show to the user the informations from that section. The presentation can be paused when needed, otherwise will contain at it's normal speed. There is also a button that will allow to activate the camera in order to take pictures. At this moment the tour will stop. After the picture is taken, the user can access the button "Back" and the tour will contain. At the end the tour cannot be reloaded. The beginning page in which you can scan a new QR will appear. When user is situated in onother page and he clicks on the logo from the header, he will be redirected to the Virtual tour page.

Photo Album Page

Here the user will find all the pictures that he tooked during the tour. He also has multiple options for share on social media on which he can select.

Information Page

The page Information contains all the exhibits that were in the museum with a breif description so that the user can access easily. In this way he doesn't have to access all the virtual tour again.

After a new museum is visitated, the new exhibits will be added here too.

Reviews Page

In the review page, user have access to reviews which belong others' users. Using Instant bot, user is able to translate each review in 104 languages, based on his preferences. Also, user is able to leave his own

review filling up the form placed in the right side of the page. He needs to leave his Name, wich is a mandatory field and the message name too. The user can leave a vote between 0 and 6 stars and when he submits the review, the review becomes visible for all the Mug's users. Until the two fiels aren't filled in, the button Submit is unenabled.

Help&Support Page

The page Help comes with a few instructions of using the application.

Contact Page

The contact page comes with some information about the ways to contact us. There is mentioned the email and the phone number. Also, users can find in the Contact page, the link to the Review page where they

can leave their feedback. The last but not the least, users can find links to the Facebook page, Twitter, Whatsapp and gmail.

QR code

The QR code will be done using The METAQRCODE API, which provides developers a way to create their own XML metadata and put it inside a QRCode. This API is Rest based, and can be consumed using

JSON or XML. We used this because it is a Single purpose API. It is easier to develop and because of the popularity of the tool, the support is more easy to find.

Data

The informations will be taken from the site data about Romanian museums and Muzee si Colectii din Romania. The data will be stored in our database, a Postgres Sql Database. We will use API to take the data

from there into our application.

Instant Translate

For the Review page we will use a bot that will translate all the text. The Instant Bot can translate anything between 104 languages. This will increase the users interaction. If the user doesn't speak the language

of another user, there is a button Translate which will display a pop up in order for the user to choose the desired language. After that the bot will change the text to that language. This will happen only for this user, the text will remain the same for the others.
